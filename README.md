## Технологический стек и архитектура

Проект разработан с акцентом на разделение логики и представления.

* **Движок:** Unity 2022.3 LTS
* **Платформа:** WebGL (Mobile Friendly)
* **Использованные инструменты:** Addressables, UniRx, Zenject, DOTween.
* **Архитектура** Использован архитектурный паттерн MVP для разделения логики и UI. Связь организована через Zenject. Окно супермаркета загружается с помощью Addressables. Создана структура HUD + WindowService как задел под большее количество окон. Для окна супермаркета использован дополнительный MonoInstaller для динамической регистрации загруженного префаба.
